# Automatizando instalações

## Install Docker
- Baixar o script em algum diretório `install-docker.sh`
- Navegar até o diretório
- Atualizar a permissão: `chmod 755 install-docker.sh`
- Rodar o script (como superusuário): `./install-docker.sh`
